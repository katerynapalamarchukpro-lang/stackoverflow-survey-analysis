# 📊 Аналіз опитування розробників Stack Overflow

* **Інструменти:** `Python`, `Pandas`, `Jupyter Notebook`
* **Опис проєкту:** Комплексний аналіз великого датасету щорічного опитування розробників від Stack Overflow. Мета — дослідити структуру вибірки, досвід роботи, формати зайнятості та рівні доходів спеціалістів.

---

### 🧠 Ключові навички та методи, використані в проєкті:
* **Робота з даними:** завантаження та дослідження великих датасетів.
* **Статистика:** обчислення мір центральної тенденції (середнє, медіана, мода).
* **EDA:** фільтрація за різними критеріями, групування та агрегація.
* **Обробка пропусків:** робота з NaN значеннями та перевірка повноти даних.
* **Аналітика:** розрахунок відсотків, перцентилів та робота з множинами.
* **Презентація:** представлення результатів у зрозумілому табличному форматі.
